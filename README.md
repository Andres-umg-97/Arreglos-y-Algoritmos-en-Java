# Arreglos y Algoritmos en Java

# Ejercicio algorítmico:

Para resolver el problema se implementó un algoritmo que recorre el arreglo de números enteros una sola vez, manteniendo variables auxiliares para almacenar el mayor, segundo mayor, menor y segundo menor valores encontrados. De esta forma, es posible determinar el segundo número mayor y el segundo número menor sin necesidad de ordenar el arreglo, cumpliendo con las restricciones establecidas. El algoritmo presenta una complejidad temporal O(n) y una complejidad espacial O(1), lo que garantiza una solución eficiente.
